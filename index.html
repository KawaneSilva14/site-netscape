A História do Netscape

De startup revolucionária a marco histórico na evolução da internet.

Abril 1994

Fundação da Mosaic Communications

Marc Andreessen e Jim Clark fundam a empresa que daria origem ao navegador Netscape, com vários ex-membros da equipe do NCSA Mosaic.

Outubro 1994

Lançamento do Netscape Navigator 1.0

O primeiro navegador comercial popular é lançado, com suporte a múltiplas plataformas e recursos inovadores para a época.

Dezembro 1994

Mudança de Nome para Netscape

A empresa muda seu nome de Mosaic Communications para Netscape Communications Corporation após disputa de marca com o NCSA.

Agosto 1995

Netscape abre capital (IPO)

As ações da empresa valorizaram de US$ 28 para US$ 58 no primeiro dia, marcando o início da bolha das empresas ponto-com.

1995-1996

Guerra dos Navegadores

Começa a competição com o Internet Explorer da Microsoft, que levaria a grandes inovações, mas também à fragmentação de padrões web.

1998

Código aberto e Mozilla

Netscape libera o código-fonte do navegador e cria o projeto Mozilla, lançando as bases para o futuro Firefox.

Novembro 1998

Aquisição pela AOL

A America Online adquire a Netscape por US$ 4,2 bilhões, numa tentativa de concorrer com a Microsoft.

2003

Fim do suporte

A AOL encerra o desenvolvimento do Netscape Navigator, com o navegador perdendo para o Internet Explorer na guerra dos navegadores.

Fevereiro 2008

Aposentadoria oficial

A AOL anuncia oficialmente o fim do suporte ao Netscape Navigator, recomendando aos usuários a migração para o Firefox.

Versões Marcantes

Conheça as versões mais importantes do navegador que mudou a história da web.

1.0

Netscape Navigator 1.0

Outubro 1994

O primeiro navegador comercial popular que introduziu uma nova era na web.

  • Suporte a múltiplas plataformas
  • Interface de usuário intuitiva
  • Recursos de segurança avançados
  • Visualização de imagens enquanto carregava
2.0

Netscape Navigator 2.0

Março 1996

Trouxe suporte a JavaScript e uma série de recursos inovadores.

  • JavaScript (inicialmente LiveScript)
  • Frames e Java applets
  • Suporte a cookies
  • Protocolo SSL para segurança
4.0

Netscape Communicator 4.0

Junho 1997

Evolução para uma suíte completa de comunicação e internet.

  • Cliente de email integrado
  • Editor HTML (Composer)
  • Suporte a camadas CSS
  • Messenger e calendário

O Legado Vive

A influência do Netscape persiste em tecnologias e conceitos que usamos até hoje.

Mozilla Firefox

O Firefox nasceu diretamente do projeto Mozilla, iniciado quando a Netscape liberou seu código-fonte em 1998, continuando o legado de inovação na web.

JavaScript

Desenvolvida pela Netscape em 1995, a linguagem JavaScript se tornou o padrão para programação web e é usada em quase todos os sites modernos.

SSL/TLS

O protocolo SSL, desenvolvido pela Netscape, evoluiu para o TLS, formando as bases da segurança na internet moderna e permitindo o e-commerce.

"Quando você está a usar métodos convencionais ou experimentais, ainda existe espaço para inovação. Alguém deve se arriscar e ser o primeiro a implementar algo."

Marc Andreessen
